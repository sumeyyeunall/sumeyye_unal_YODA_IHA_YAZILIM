###Data Types
Değişkenin ne tür bir veri (sayı, metin, liste vb.) tuttuğunu belirten sınıflandırmalardır. Python dinamik bir dil olduğundan bunları otomatik olarak algılar 

int (Integer):Tam sayılar
float (Floating Point): Ondalıklı sayılar
str (String): Metin verileri, tek tırnak veya çift tırnak içerisinde tanımlanırlar
bool (Boolean): Mantıksal veri tipidir yani yalnızda iki değer alabilir:True veya False. Koşullu ifadelerde kullanılır genellikle.

girdi:
```
x, y = 10, 3
division = x/y
summation = x+y
print(f"Data type of x: {type(x)}, y: {type(y)}, division: {type(division)}, summation: {type(summation)}")
```
çıktı:
```
Data type of x: <class 'int'>, y: <class 'int'>, division: <class 'float'>, summation: <class 'int'>
```

tuple:Sıralı ve immutable bir veri koleksiyonudur yani değiştirilemez ve güncellenemez. () ile tanımlanır. Programda yanlışlıkla değiştirilmemesi gereken veriler(koordinatlar vs.) için kullanılır.
girdi:
```
my_tuple = (1, 2, 3)
nums_tuple = tuple(liste)
```

dict(Sözlük):Sırasız bir key-value çiftinden oluşan koleksiyondur. {} ile tanimlanır. Listelerdeki indexler yerine unique keyler kullanılır. Veriye ulaşımı hızlıdır. 
girdi:
```
address = {
    "City": "Bonn",
    "Street": "Nussallee",
    "HouseNumber": 15
}
print("HouseNumber:", address["HouseNumber"])
address["LectureRoom"] = 14
print("Keys:", address.keys())
print("Values:", address.values())
```
çıktı:
```
HouseNumber: 15
Keys: dict_keys(['City', 'Street', 'HouseNumber', 'LectureRoom'])
Values: dict_values(['Bonn', 'Nussallee', 15, 14])
```

Conditionals(if/elfi/else):Programın akışını belirli bir koşulun True veya False olmasına göre yönlendiren karar verme yapılarıdır.
if:koşulu kontrol eder ve doğruysa altındaki kod bloğu çalışır.
elif(else if):önceki if veya elif yanlışsa yeni bir koşulu kontrol eder.
else:yukarıdaki koşulların hiçbiri doğru değilse çalışır.

Döngüler(For ve While):Bir kod bloğunu belirli bir sayıda veya belirli bir koşul sağlandığı sürece tekrar tekrar çalıştırmak için kullanılır.

for:liste, tuple, dict, string veya range() fonksiyonundaki her bir eleman üzerinde sırayla gezinir.

while:belirli bir koşul True olduğu sürece çalışmaya devam eder koşul False olduğunda durur. Döngüdeki false koşulu sağlanmadığı sürece sonsuz döngüye girer. Break ile döngü hemen sonlandırılırken, continue ile o adım atlanır ve bir sonraki adıma geçilir. 
girdi:
```
nested_list = [
    [1, 2, 3],
    [4, 0, 9],
    [7, 8]
]

#nested listin uznulugu
print("Length of nested_list:", len(nested_list))
#her inner listin uzunlugu
for inner_list in nested_list:
    print("Length of inner list:", len(inner_list))
#--
total_sum = 0
for inner_list in nested_list:
    for element in inner_list:
        if element == 0:
            print("Found zero element")
        total_sum += element
print("Sum of all elements:", total_sum)
```
çıktı:
```
Length of nested_list: 3
Length of inner list: 3
Length of inner list: 3
Length of inner list: 2
Found zero element
Sum of all elements: 34
```

Functions(Fonksiyonlar): Belirli bir görevi yerine getirmek için tasarlanmış, yeniden kullanılabilir kod bloklarıdır.def anahtar kelimesi ile tanımlanırlar. Kodun yönetilmesini kolaylaştırır küçük parçalara bölerek ve tekrar kullanımı olduğundan code repeati azaltır.
girdi:
```
#örneğin utilities.py dosyasina bu fonksiyonu kaydedelim
def sum_nested_list(nested_list):
    total = 0
    for inner in nested_list:
        for value in inner:
            total += value
    return total
```
```
#main.py içerisinde 
from utilities import sum_nested_list
data = [
    [1, 2, 3],
    [4, 5],
    [6]
]
print(sum_nested_list(data))
```
çıktı:
```
21
```

Parametreler(Parametre vs Argüman):Fonksiyonu tanımlarken (def) belirttiğimiz değişkenlere parametre(örn: nested_list), fonksiyonu çağırırken(sum_nested_list(another_list)) içine verdiğimiz gerçek değere argüman(örn: another_list) denir.
